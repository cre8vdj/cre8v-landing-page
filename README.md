# 🌐 Responsive Landing Page — HTML / CSS / JS

![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-F7DF1E?logo=javascript) ![License](https://img.shields.io/badge/License-MIT-green) ![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen)

A **fully responsive single-page portfolio landing page** built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies. Features smooth scrolling, animated sections, a sticky navigation bar, and a working contact form.

🔗 **Live Demo:** [https://cre8vdj.github.io/cre8v-landing-page/](https://cre8vdj.github.io/cre8v-landing-page/)

---

## ✨ Features

- 🖥️ Fully **responsive** — works on desktop, tablet, and mobile
- 👀 **Sticky navigation** bar with scroll-aware highlighting
- 🎨 **Hero section** with animated headline and CTA button
- 📌 **About**, **Skills**, **Projects**, and **Contact** sections
- ✨ **CSS animations** and smooth scroll transitions
- 📱 **Mobile hamburger menu** for small screens
- ✅ **Contact form** with client-side validation
- 🎨 Clean, modern design — easy to customize
- ⚡ Zero dependencies — pure vanilla HTML/CSS/JS

## 🌐 Deployment (GitHub Pages)

This site is deployed automatically via GitHub Pages from the `main` branch.

To deploy your own fork:
1. Go to **Settings → Pages**
2. Set Source to `main` branch, `/ (root)`
3. Save — your site will be live at `https://<username>.github.io/<repo-name>/`
